## 🛠️ Sobre o Projeto
O **SmartReport** é um ecossistema Full Stack desenvolvido para resolver um problema real de gestão: a demora em extrair insights de grandes volumes de dados operacionais. Utilizando a **Llama-3 (Groq API)** e o poder do **Pandas**, o sistema automatiza o saneamento, a análise estatística e a geração de documentos técnicos.

### 🌟 Funcionalidades Principais:
* **Upload Inteligente**: Suporte a arquivos `.csv` e `.xlsx` via Drag & Drop.
* **Data Sanitization**: Mascaramento automático de dados sensíveis antes do envio para a IA.
* **Insights de IA**: Geração de resumos técnicos, análise de KPIs e recomendações estratégicas.
* **Visualização Dinâmica**: Gráficos de médias gerados em tempo real com Matplotlib.
* **Relatório PDF Multi-página**: Geração de documentos profissionais prontos para impressão/download.

## 🏗️ Arquitetura e Tech Stack



### **Frontend** (React + TypeScript)
* **Vite**: Build ultra-rápido.
* **Tailwind CSS**: Interface moderna e responsiva.
* **Axios**: Comunicação com a API.
* **Lucide-react**: Ícones elegantes.

### **Backend** (Python + FastAPI)
* **Pandas**: Motor de processamento de dados.
* **Groq API (Llama-3)**: O cérebro por trás da análise de dados.
* **ReportLab**: Motor de geração de PDFs multi-páginas.
* **Matplotlib**: Renderização de gráficos estatísticos.
